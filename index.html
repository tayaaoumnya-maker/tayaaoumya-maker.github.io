SMMIA = Social Media Marketing + Intelligence Artificielle

💭 Philosophie

Créer des agents qui travaillent pour vous pendant que vous vivez votre vie.

⚙️ Concepts Techniques Fondamentaux

IA Navigateur vs Terminal

  • Navigateur : Performances limitées, contexte restreint
  • Terminal : Connexion directe au processeur, puissance maximale
  • Analogie : moteur thermique vs moteur électrique

Claude CLI

Version IA exécutée dans le terminal.

  • Plus puissante que navigateur
  • Permet automatisations avancées
  • Installation : Node.js → Git → Claude CLI

Open CL

Assistant IA accessible depuis mobile (WhatsApp, Discord…).

  • Accès temps réel à l’ordinateur
  • Exécution de tâches à distance
  • Ordinateur doit rester allumé (VPS recommandé)

VPS

Serveur virtuel loué (10–15€/mois) permettant de garder les automatisations actives 24h/24.
