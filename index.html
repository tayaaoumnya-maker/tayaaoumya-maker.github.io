Introduction

Contexte & Objectifs

Contexte

Live de formation sur l'automatisation business utilisant l'intelligence artificielle.

Objectif principal

Comprendre l'automatisation et générer des revenus ("faire de l'oseille").

Concept clé

SMMIA (Social Media Marketing Intelligence Artificielle) = combinaison entre la vente de services (SMMA) et l'intelligence artificielle pour automatiser la prestation de services.

Philosophie du cours

"L'objectif c'est que vos agents bossent pour vous. Vous êtes à la chicha, vous savez que derrière ça fait rentrer de l'oseille"

Concepts Techniques Fondamentaux

1. IA sur Navigateur vs Terminal

Comparaison

Tableau résumé :

  • Navigateur : performance bridée, ressources partagées, contexte fenêtre limité (onglets ouverts).
  • Terminal : performance boostée, connexion directe au processeur, aucune limitation de contexte.

Analogie : moteur thermique (lent) vs moteur électrique (instantané). Principe : "Quand vous passez sur le terminal, l'IA c'est processeur IA."

2. Claude CLI

Définition & Avantage

Version de l'IA Claude exécutée directement dans le terminal. Plus puissant que l'utilisation via navigateur et permet de créer sections Shopify, automatisations, etc.

Installation Windows (résumé)

  • Installer Node.js
  • Installer Git
  • Installer Claude CLI
  • Documentation disponible dans l'espace replay
3. Open CL

Caractéristiques

Assistant IA accessible depuis mobile (WhatsApp, Discord, Telegram) avec accès en temps réel à tout l'ordinateur et aux skills/agents configurés.

Exemple concret : "Je suis en closing... 'S'il te plaît Tasha, fais-moi une facture de 1200 € pour un site vitrine plus un système de réservation.' Il le fait, j'ai directement tout sur mon téléphone."

Pré-requis : l'ordinateur doit rester allumé (solution recommandée : VPS).

4. VPS (Virtual Private Server)

Définition & Coût

Ordinateur virtuel loué à distance. Coût typique : 10–15€ par mois. Avantage : reste allumé 24h/24, permet de faire tourner les automatisations. Alternative : ordinateur physique qui reste allumé.

5. Skills (Compétences IA)

Définition & Méthode

Instructions spécialisées mémorisées par l'IA pour effectuer des tâches précises.

Types mentionnés : Mobile First (sites responsive), Front-end Design, UI/UX, Facture (génération automatique).

Méthode de création

  1. Réaliser la tâche complète avec l'IA (ex : créer une facture de A à Z).
  2. Donner l'instruction : "Maintenant je veux que à chaque fois que je te demande une facture, tu me refais exactement la même facture en modifiant selon mes variables".
  3. L'IA crée automatiquement le skill.
6. N8N - Outil d'automatisation

Présentation

N8N = tableau blanc où on connecte des blocs entre eux pour créer des workflows automatisés.

Chiffres clés :

  • 70 000 étoiles GitHub
  • Plus de 200 000 utilisateurs actifs
  • 1286 nodes disponibles

Architecture / concepts

  • Trigger : événement qui lance le workflow (ex: horaire, mail reçu, nouvelle commande Shopify).
  • Node : bloc d'action (envoyer mail, scraper, poster sur réseaux, appeler API).
  • Workflow : enchaînement d'actions automatisées.

7. MCP (Model Context Protocol)

Fonction & Exemple

Définition simple : "Une prise, un connecteur". Transforme Claude en expert instantané d'un outil spécifique (ex : MCP N8N fait connaître les 1286 nodes à l'IA).

Avant / Après MCP (résumé)

  • Apprentissage : avant → lire la doc ; avec MCP → aucun apprentissage nécessaire.
  • Erreurs (hallucinations) : avant → fréquentes ; avec MCP → quasi zéro.
  • Temps de création : avant → ~45 minutes ; avec MCP → ~3 minutes.
  • Couverture : avec MCP → 99% des propriétés.

Autres MCP disponibles : MCP Notion (envoi automatique de projets vers Notion), et d'autres pour de nombreux outils.

Utilisation N8N avec Claude

Workflow résumé

  1. Décrire l'automatisation souhaitée à Claude (vocal ou texte).
  2. Claude génère le workflow JSON.
  3. Importer le workflow dans N8N.
  4. Lancer (Start).

Commande simple : dire à Claude "Installe-moi le MCP N8N". Résultat : Claude devient automatiquement expert N8N.

Recommandation : "Installez le MCP N8N complet, il va tout vous faire".

  3. L'IA mémorise automatiquement le skill.

Business Plan : Prospection Automatisée

Cas d'étude : Sites vitrines pour plombiers

Principe

Identifier des artisans sans site web, créer automatiquement un site personnalisé, et les démarcher avec la maquette prête.

Workflow complet (7 étapes)

  1. Configuration du filtre : critère recherche "Plombier Paris", limite 20 prospects.
  2. Scraping : Outcraper → extraire fiches Google My Business (filtre : fiche GMB existe MAIS pas de site internet). Données récupérées : nom, adresse, email.
  3. Génération du site : envoyer les données à Claude/Gemini via API. Prompt : "Génère un site vitrine HTML complet et moderne pour un plombier" — personnalisation automatique (nom, adresse, coordonnées).
  4. Hébergement sur GitHub : code HTML enregistré automatiquement dans un repository.
  5. Déploiement sur Vercel : push automatique → URL de prévisualisation fonctionnelle.
  6. Rédaction email de prospection : composition automatique d'un mail personnalisé (ex : proposition à 200€ avec lien maquette).
  7. Envoi automatique : via intégration Gmail dans N8N. Utiliser boîte email personnelle pour éviter les spams.
Résultats concrets
  • ✅ Cas 1 : Personne contactée par Pages Jaunes dès le lendemain pour référencement prioritaire.
  • ✅ Cas 2 : Entrepreneur américain générant 2 000$ avec cette méthode (Twitter).
  • ✅ Validation : Le formateur a prédit la viralisation avant celle-ci aux États‑Unis.

Cas d'Usage Concrets

Applications réussies
  • Création de sections Shopify : utilisation de Claude pour coder des éléments e‑commerce.
  • Designs avec Gemini : génération d'éléments visuels et graphiques.
  • Assistant de closing mobile : générer une facture de 1200€ pendant un rendez‑vous client.
  • Skill Facture mémorisé : format reproduit automatiquement à chaque demande.
  • Prospection artisans automatisée : workflow complet de génération de leads.
  • Gestion d'avis automatique : réponses automatisées aux avis Google My Business.
  • Contrôle d'ordinateur à distance : commander applications et fichiers depuis mobile.
  • SMMIA : agence de marketing entièrement automatisée.
Workflow alternatif (extrait)

Exemple : "Créer une automatisation : quand je reçois un mail après 16h, envoyer des messages à tout le monde et enregistrer les messages dans un Google Sheet".

Points clés à retenir

Résumé
  • Nouvelle mentalité business : avant de lancer, se demander "Est‑ce que je peux l'automatiser ?"
  • Avantage temporel : tester un business en 1 heure au lieu d'y consacrer 100% du temps.
  • Risque d'hallucination : sans MCP, l'IA peut inventer des propriétés inexistantes et créer des workflows non‑fonctionnels.
  • Performance Terminal : connexion directe au processeur = IA plus puissante qu'en navigateur.
  • MCP = Expert instantané : accès à documentation et expertise sans apprentissage.
  • Rentabilité : "Si vous pouvez l'automatiser, c'est que l'idée business est rentable."
