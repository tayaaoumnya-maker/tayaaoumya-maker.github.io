Claude CLIUtilisation de Claude via le terminal. Plus performant que via le navigateur car connexion directe « Processeur ↔ IA ».

1.3 — Concepts Automatisation (N8N)

ConceptDéfinition
N8NPlateforme d'automatisation (no-code/low-code) pour créer des workflows. Large écosystème de nœuds.
Noeuds (Nodes)Blocs d'actions (envoyer email, scraper, poster, appeler API…).
TriggerÉvénement déclencheur (paiement, nouvel email, horaire).

PARTIE 3 — Le « BANGER » : Business Plan détaillé

Présentation principale : Workflow N8N entièrement automatisé pour vendre des sites web en volume à des artisans (ex : plombiers).

Étape 1 — Identification des prospects — Scraping

  • Outil : Outscraper (API)
  • Source : Google Maps
  • Cible : entreprises avec fiche GMB MAIS sans site web
  • Filtre : entreprises SANS site web (ex : plombiers à Paris — 20 prospects en demo)

Étape 2 — Création du site vitrine automatique

  • IA : Claude ou Gemini génère une landing page HTML moderne et personnalisée
  • Injection automatique des coordonnées réelles du prospect

Étape 3 — Déploiement

  1. Envoyer le code HTML sur GitHub (push automatique)
  2. Déployer sur Vercel → URL créée automatiquement
  3. Résultat : maquette en ligne prête à montrer

Étape 4 — Prospection automatisée — Cold Email

  • Email personnalisé envoyé automatiquement au prospect
  • Inclure le lien de la maquette + proposition commerciale (200 €)
  • Conseil anti-spam : utiliser une boîte mail personnelle connectée au workflow

Schéma synthétique du workflow

Outscraper (Google Maps)
      ▼
Claude / Gemini (Génération HTML)
      ▼
GitHub → Vercel (URL en ligne)
      ▼
Cold Email auto (lien démo)
      ▼
Vente (200 € par site)

PARTIE 5 — Erreurs à éviter et points d'attention

Erreurs techniques

  • Ne pas utiliser l'IA via navigateur pour tâches complexes — utiliser CLI.
  • Ne pas travailler sans MCP — risque d'hallucinations (sans MCP = 45 min, avec MCP = 3 min).
  • Ne pas négliger le VPS — automatisations arrêtent si PC éteint.

Erreurs de prospection

  • Mal filtrer les données Outscraper — cibler uniquement entreprises SANS site web.
  • Mal configurer accès API — donner les accès à Claude pour configurer (GitHub, Vercel…).
  • Utiliser service d'envoi générique → risk spam ; utiliser boîte personnelle liée au workflow.

Erreurs d'apprentissage & navigation

  • Essayer d'apprendre toute la doc N8N manuellement — préférer config pilotée par IA.
  • Ne pas voir canaux Discord — cliquer sur « Afficher tous les salons »
