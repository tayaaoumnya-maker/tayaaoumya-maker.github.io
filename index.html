Concepts Techniques Fondamentaux

1. IA sur Navigateur vs Terminal

Critère IA sur Navigateur IA sur Terminal
Performance Bridée, partage ressources processeur Boostée, connexion directe
Limitations Contexte fenêtre limité, autres onglets ouverts Aucune limitation
Analogie Moteur thermique (lent) Moteur électrique (instantané)
Connexion Navigateur → IA Terminal → Processeur → IA
« Là quand vous passez sur le terminal, l'IA ça fait processeur IA. C'est comme quand on parle voiture [...] c'est terminal processeur. » Principe

2. Claude CLI (Command Line Interface)

Définition
Version de l'IA Claude exécutée directement dans le terminal
Avantage
Plus puissant que l'utilisation via navigateur
Utilisation
Permet de créer des sections Shopify, des automatisations, etc.

Installation Windows :

  1. Installer Node.js
  2. Installer Git
  3. Installer Claude CLI
  4. Documentation disponible dans l'espace replay

3. Open CL

Caractéristiques :

  • Assistant IA accessible depuis mobile (WhatsApp, Discord, Telegram)
  • Accès en temps réel à tout l'ordinateur
  • Accès aux skills et agents configurés
« Je suis en closing, dans un rendez-vous avec un client : ‘S'il te plaît Tasha, fais-moi une facture de 1200 € pour un site vitrine plus un système de réservation.’ Il le fait, j'ai directement tout sur mon téléphone. » Exemple d'usage concret

Prérequis : L'ordinateur doit rester allumé (solution : VPS)

4. VPS (Virtual Private Server)

Définition
Ordinateur virtuel loué à distance
Coût
10–15 € par mois
Avantage
Reste allumé 24h/24, permet de faire tourner les automatisations
Alternative
À l'ordinateur physique qui doit rester allumé

5. Skills (Compétences IA)

Définition
Instructions spécialisées mémorisées par l'IA pour effectuer des tâches précises

Types de skills mentionnés :

  • Mobile First : Sites web responsive mobile
  • Front-end Design : Création de beaux designs automatiquement
  • UI/UX : Navigation intuitive optimisée
  • Facture : Génération automatique de factures

Méthode de création :

  1. Réaliser une tâche complète avec l'IA (ex : créer une facture de A à Z)
  2. Donner l'instruction : « Maintenant je veux que à chaque fois que je te demande une facture, tu me refais exactement la même facture en modifiant par rapport à ce que je veux »
  3. L'IA crée automatiquement le skill

6. N8N — Outil d'automatisation

Définition
Tableau blanc où on connecte des blocs entre eux pour créer des workflows automatisés

Chiffres clés :

  • 70 000 étoiles GitHub
  • Plus de 200 000 utilisateurs actifs
  • 1 286 nodes (blocs) disponibles

Architecture :

Trigger (Déclencheur)
Événement qui lance le workflow (ex : « tous les jours à 8h », « quand je reçois un mail », « nouvelle commande Shopify »)
Node (Nœud)
Bloc d'action individuel (envoyer mail, scraper site, poster réseaux sociaux, appeler API)
Workflow
Enchaînement automatisé de A à Z

Comparaison avec concurrents :

Outil Avantages Type
N8N Gagne « haut la main », plus flexible Open source
Make Solution payante SaaS
Zapier Similaire à Make SaaS

7. MCP (Model Context Protocol)

Définition simple
« Une prise, un connecteur »
Fonction
Transforme Claude en expert instantané d'un outil spécifique

Exemple MCP N8N :

  • Claude connaît les 1 286 nodes par cœur
  • Sait exactement quel branchement faire à quel moment
  • Devient un expert de l'automatisation

Comparaison Avant / Après MCP :

Critère Avant MCP Avec MCP
Apprentissage Toute la documentation N8N à mémoriser Aucun
Erreurs Fréquentes, hallucinations Zéro hallucination
Temps création 45 minutes pour workflow basique 3 minutes pour workflow complet
Couverture Partielle 99 % des propriétés
Méthode Regarder 4 vidéos YouTube par action Description en français

Autres MCP disponibles :

  • MCP Notion : Envoi automatique de projets vers Notion
  • MCP disponibles pour de nombreux outils

Business Plan : Prospection Automatisée

Cas d'étude : Sites vitrines pour plombiers

Principe : Identifier des artisans sans site web, créer automatiquement un site personnalisé, et les démarcher avec la maquette prête.

Workflow complet en 7 étapes

Étape 1 — Configuration du filtre

  • Critères de recherche : « Plombier Paris »
  • Limite de résultats : 20 prospects

Étape 2 — Scraping avec Outcraper

  • Utilisation de l'API Outcraper
  • Extraction des fiches Google My Business
  • Filtre spécifique : Plombiers qui ont une fiche Google MAIS pas de site internet
  • Données récupérées : nom entreprise, adresse, email

Étape 3 — Génération automatique du site

  • Envoi des données à Claude ou Gemini via API
  • Prompt : « Génère un site vitrine HTML, complet et moderne pour un plombier »
  • Personnalisation automatique avec les données extraites (nom, adresse, coordonnées)

Étape 4 — Hébergement sur GitHub

  • Code HTML automatiquement enregistré dans un repository GitHub

Étape 5 — Déploiement sur Vercel

  • Push automatique du repository sur Vercel
  • Génération d'une URL de prévisualisation fonctionnelle

Étape 6 — Rédaction de l'email de prospection

  • Composition automatique d'un mail personnalisé
  • Contenu type : « Je me permets de vous contacter car j'ai remarqué que [nom entreprise] n'a pas encore de site Internet alors que vos concurrents à Paris en ont déjà un. J'ai pris l'initiative de créer une maquette de site vitrine. Ce site est entièrement personnalisé avec vos coordonnées pour 200 €. »

Étape 7 — Envoi automatique

  • Envoi via l'intégration Gmail dans N8N
  • Email contenant le lien vers la maquette Vercel
  • Utilisation de la boîte email personnelle pour éviter les spams

Points Clés à Retenir

  • 💡 Nouvelle mentalité business : Avant de lancer un business, se poser la question « Est-ce que je peux l'automatiser ? »
  • ⏰ Avantage temporel : Possibilité de tester un business plan en 1 heure au lieu de s'y consacrer à 100 %
  • 🚫 Risque d'hallucination : Sans MCP, l'IA peut inventer des propriétés inexistantes et créer des workflows non-fonctionnels
  • 🚀 Performance Terminal : Connexion directe au processeur = IA significativement plus puissante qu'en navigateur
  • 🔌 MCP = Expert instantané : Les MCP donnent accès à toute la documentation et expertise d'un outil sans apprentissage
  • 💰 Rentabilité : « Si vous pouvez l'automatiser, c'est qu'elle [l'idée business] est rentable. »
